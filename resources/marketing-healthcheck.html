品牌基礎

0/10
  • 目標客群明確能用一句話描述理想客戶的樣貌與痛點
  • 品牌核心價值清晰品牌代表什麼？與競品的核心差異是什麼？
  • 品牌故事完整有一個能引起受眾共鳴的創業 / 品牌故事
  • 視覺識別統一Logo、品牌色、字體有明確規範且跨平台一致
  • 品牌語調一致所有平台的文案風格統一（專業 / 親切 / 幽默）
  • USP（獨特賣點）明確客戶選你而不選競品的關鍵理由是什麼？
  • 品牌定位語有一句精準的品牌 Slogan 或定位聲明
  • 競品分析了解前 3 大競品的優劣勢與行銷手法
  • 品牌接觸點一致名片、官網、社群、包裝的品牌體驗一致
  • 客戶見證 / 社會證明有收集並展示客戶好評、合作案例
品牌基礎是一切行銷的根基。如果這區得分偏低，建議優先補強，才能讓後續的行銷投入事半功倍。
2

社群經營

0/10
  • 選對主力平台有出現在目標客群最活躍的 1-2 個平台
  • 發文頻率穩定每週至少發布 3-5 篇社群內容
  • 內容類型多元教學、幕後花絮、互動、促銷等類型輪流發布
  • 短影音 / Reels 經營有定期發布 Reels、TikTok 或 Shorts 內容
  • 限時動態活躍每天有更新 Stories，保持品牌存在感
  • 互動回覆即時24 小時內回覆留言與私訊
  • Hashtag 策略有效使用與品牌 / 受眾相關的標籤組合
  • 每篇有 CTA每篇貼文都有引導行動（留言、點連結、分享）
  • 社群數據定期檢視有追蹤觸及率、互動率、粉絲成長數據
  • 跨平台導流有策略地將流量從其他平台導向主力社群
社群的價值不在粉絲數量，而在互動品質。100 個會留言的粉絲 > 10,000 個沈默的追蹤者。
3

內容策略

0/10
  • 內容日曆排程有每月 / 每週的內容規劃表
  • 內容主題明確有 3-5 個固定的內容主軸 / 系列
  • SEO 基礎優化網站有做標題、描述、關鍵字的基礎 SEO
  • 長篇內容經營有定期發布部落格、知識庫等深度內容
  • 內容再製策略一篇內容會改編為不同格式（文字→影片→圖卡）
  • UGC 用戶生成內容有鼓勵客戶分享使用心得或標記品牌
  • 名單收集機制有在收集潛在客戶的聯絡方式（Email / LINE）
  • 內容漏斗完整內容涵蓋認知→考慮→決策各階段
  • 爆款內容模式了解自己哪類內容表現最好，並持續放大
  • 內容價值導向內容以提供價值為主，而非純推銷
好的內容策略讓每篇內容都有目的：吸引新客、培養信任、或推動轉換。三者缺一不可。
4

廣告與轉換

0/10
  • 有投放數位廣告有在 Meta、Google 或其他平台投放廣告
  • 受眾設定精準廣告受眾有根據數據設定，而非亂槍打鳥
  • 素材定期更新廣告素材至少每 2-4 週更換一次
  • A/B 測試有對文案、圖片、受眾進行測試比較
  • 轉換追蹤設定有安裝 Pixel 或轉換 API 追蹤成效
  • ROAS 追蹤有追蹤廣告投資報酬率，並持續優化
  • Landing Page 優化廣告到達頁有針對轉換進行設計
  • 再行銷策略有對曾互動的受眾進行再行銷投放
  • 漏斗分層投放有區分冷受眾、暖受眾、熱受眾的投放策略
  • 預算分配合理廣告預算有根據成效數據動態調整
廣告是放大器，不是救世主。品牌基礎不穩、內容品質差，再多預算也換不到好成效。
5

數據與優化

0/10
  • GA / 網站分析網站有安裝 Google Analytics 並定期查看
  • 社群 Insights 定期分析每週查看社群平台的數據洞察報告
  • 轉換漏斗追蹤知道從曝光到成交各階段的轉換率
  • 客戶來源歸因了解客戶從哪個管道認識你、如何決策
  • 月度成效報表每月整理行銷成效報告，追蹤 KPI
  • 數據驅動決策行銷策略調整是根據數據而非直覺
  • 競品數據監測有定期觀察競品的行銷動態與表現
  • 內容表現排名知道哪些內容效果最好，並能複製成功
  • 客戶回饋收集有系統性地收集客戶意見與滿意度
  • 定期策略檢討至少每季調整一次行銷策略方向
沒有追蹤就沒有優化。養成看數據的習慣，是從「憑感覺做行銷」邁向「科學化行銷」的關鍵。
